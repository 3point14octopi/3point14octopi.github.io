Josephine Flynn's Online Portfolio

Hello! My name is Josephine and I'm a game programmer from the Greater Toronto Area. Rather than programming gameplay mechanics, I like to focus on developing modular solutions for larger systems, and tying the underlying pieces of games together.
While systems are my favourite things to develop, I still enjoy learning other disciplines and tinkering with new languages or frameworks. I've done some graphics programming using OpenGL and the GLSL language, written a few applications using .NET's Socket library, and am currently using this page as a way to brush up on my HTML and CSS!
For the last three years I've been a part of JAFBO, a student-run game studio, and our games can be seen below. I've also showcased a few of my personal projects that I think are particularly interesting.
